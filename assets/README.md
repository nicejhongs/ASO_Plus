# 사운드 파일

이 폴더에 다음 wav 파일들을 넣어주세요:

## 필요한 파일

1. **shoot.wav** - 총알 발사 소리
   - 짧고 경쾌한 소리 (권장: 0.1~0.3초)
   - 예: 레이저 발사음, 탄환 소리

2. **explosion.wav** - 폭발 소리
   - 중저음의 폭발음 (권장: 0.2~0.5초)
   - 예: 폭발음, 충돌음

## 파일이 없을 경우

wav 파일이 없으면 Windows 기본 비프음이 재생됩니다.

## 파일 형식

- 형식: WAV (PCM)
- 샘플레이트: 44100 Hz 권장
- 비트레이트: 16-bit 권장
- 채널: 모노 또는 스테레오
